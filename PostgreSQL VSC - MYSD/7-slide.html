C. Escribir y Ejecutar Consultas en VS Code
PASO 01
Abre el .sql
Archivo existente o Ctrl+N → guarda como .sql
scripts/consulta.sql
PASO 02
Escribe la consulta
Sin autocompletado. Línea a línea, con comprensión total de cada cláusula.
Filosofía Vainilla
PASO 03
Selecciona el texto
Solo la consulta a ejecutar. No selecciones todo el archivo.
Click + Shift o Ctrl+A parcial
PASO 04
Ejecuta
Ctrl+Shift+E → Run Selected Query sobre tu conexión activa
Ver resultado en panel
⚡ REGLA DORADA
NUNCA uses ejecutar todo sin revisar línea por línea lo que va a correr en el servidor compartido. Un DELETE o DROP accidental no tiene vuelta atrás.
TIPS DE EFICIENCIA EN VS CODE
Usa Ctrl+` para abrir el terminal integrado mientras tienes el editor SQL abierto
Puedes tener múltiples pestañas .sql abiertas — una por entidad o funcionalidad
Separa consultas con ; en el mismo archivo y selecciona solo la que quieres ejecutar
F5 en VS Code no ejecuta SQL — siempre selecciona + Ctrl+Shift+E
