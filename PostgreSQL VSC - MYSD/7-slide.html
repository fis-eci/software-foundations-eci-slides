FLUJO DE TRABAJO
C. Escribir y Ejecutar Consultas en VS Code
PASO 01
Abre el .sql
File existente o Ctrl+N → guarda como .sql
PASO 02
Escribe la consulta
Sin autocompletado. Línea a línea, consciente.
PASO 03
Selecciona el texto
Solo la consulta a ejecutar, no todo el archivo
PASO 04
Ejecuta
Ctrl+Shift+E o clic derecho → Run Selected Query
⚡ REGLA DORADA
NUNCA uses ejecutar todo sin revisar línea por línea lo que va a correr en el servidor compartido.
