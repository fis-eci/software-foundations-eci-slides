EXTENSIONES
Instalar SQLTools + Driver PostgreSQL
BASE
SQLTools
por Matheus Teixeira
Gestión de conexiones, escritura y ejecución manual de consultas SQL en VS Code. Es la base sobre la que se monta el driver de PostgreSQL.
COMPLEMENTO
SQLTools PostgreSQL/Cockroach Driver
extiende SQLTools
Habilita la conexión específica con bases de datos PostgreSQL desde SQLTools. Sin este driver, SQLTools no reconoce el motor.
1
Ctrl+Shift+X
abre el panel de extensiones
2
Busca "SQLTools"
instala el de Matheus Teixeira
3
Busca "SQLTools PostgreSQL"
instala el driver
4
Recarga VS Code
cuando lo solicite
Antes de continuar, aplica la configuración Vainilla para MYSD según la Guía VS Code Vainilla del curso.
