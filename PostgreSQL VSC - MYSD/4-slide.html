Instalar SQLTools + Driver PostgreSQL
BASE
SQLTools
por Matheus Teixeira
Gestión de conexiones, escritura y ejecución manual de consultas SQL en VS Code. Base sobre la que se monta el driver de PostgreSQL.
  • Conexiones guardadas y reutilizables por proyecto
  • Editor SQL con resaltado de sintaxis PostgreSQL
  • Panel de resultados integrado (grid + raw)
  • Historial de consultas ejecutadas
COMPLEMENTO
SQLTools PostgreSQL/Cockroach Driver
extiende SQLTools
Habilita la conexión específica con bases de datos PostgreSQL. Sin este driver, SQLTools no reconoce el motor pg.
  • Protocolo nativo pg_protocol (libpq compatible)
  • Soporte para conexiones SSL/TLS
  • Compatible con PostgreSQL 12–17
  • Exploración de esquemas, tablas y columnas
