VERIFICAR QUE EL SERVICIO ESTÁ ACTIVO
# Windows — buscar en Servicios:
services.msc  →  postgresql-x64-17  estado: En ejecución

# Línea de comandos (psql):
psql -U postgres          ← debe mostrar prompt postgres=#
psql -U postgres -l       ← lista todas las bases de datos
INSTALAR POSTGRESQL LOCAL
1
Descargar de postgresql.org/download (versión 16 o 17, multiplataforma)
2
Ejecutar el instalador → anotar la contraseña del usuario postgres (no la pierdas)
3
Verificar el servicio: services.msc → postgresql-x64-XX en estado Running
4
Crear la conexión en SQLTools con los parámetros de la tabla izquierda
5
Crear tu BD de práctica: CREATE DATABASE mi_proyecto;
