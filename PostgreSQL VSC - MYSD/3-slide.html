La Filosofía Vainilla: Programación Consciente
En IS-FUNDAMENTOS, VS Code se configura para DESACTIVAR la asistencia automática de IA.
El objetivo: aprender a escribir SQL línea a línea con comprensión total de cada instrucción.
3 REGLAS VAINILLA PARA MYSD
Sin autocompletado SQL avanzado — Desactivar en la configuración de SQLTools
Sin asistentes visuales de consultas — No usar generadores de queries
Ejecución siempre manual — Ctrl+Shift+E, nunca "ejecutar todo"
LO QUE NO SE HACE
  • GitHub Copilot generando el SELECT
  • Botón "Run All" sin revisar las sentencias
  • Asistentes de construcción de tablas visuales
  • Autocompletado agresivo de nombres de columnas
