FILOSOFÍA
La Filosofía Vainilla: Programación Consciente
En IS-FUNDAMENTOS, VS Code se configura para DESACTIVAR la asistencia automática de IA.
El objetivo: aprender a escribir SQL línea a línea con comprensión total de cada instrucción.
3 REGLAS VAINILLA PARA MYSD
Sin autocompletado SQL avanzado — Desactivar en la configuración de SQLTools. Cada campo y tabla se escribe a mano, con propósito.
Sin asistentes visuales de consultas — Prohibido usar generadores de queries, drag-and-drop o constructores automáticos de JOIN.
Ejecución siempre manual y selectiva — Seleccionar la consulta exacta y presionar Ctrl+Shift+E. Nunca "ejecutar todo".
LO QUE NO SE HACE
  • GitHub Copilot o Gemini generando el SELECT completo
  • Botón "Run All" sin revisar las sentencias primero
  • Asistentes visuales para construir tablas o relaciones
  • Autocompletado agresivo de nombres de columnas
  • Copiar y pegar SQL sin entender qué hace cada cláusula
  • Ejecutar todo el archivo .sql sin selección específica
Configuración detallada en: Guía VS Code Vainilla → sección MYSD (PostgreSQL). Aplica estos ajustes antes de continuar. El atajo de ejecución manual es siempre Ctrl+Shift+E — sobre la consulta seleccionada, nunca sobre el archivo completo.
