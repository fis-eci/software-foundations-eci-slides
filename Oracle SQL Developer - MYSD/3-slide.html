MÓDULO B: INSTALACIÓN DE SOFTWARE

Instalación SQL Developer

PASOS PRINCIPALES

01

Descargar

Visita oracle.com. Busca la sección de descargas de "SQL Developer". Solo requieres una cuenta gratuita básica para iniciar la descarga.

Disponible para Windows, macOS y Linux

02

Descomprimir y Ejecutar

Es una aplicación portable (no requiere instalación tradicional). Extrae el archivo en una ruta corta y sin espacios, luego ejecuta el archivo principal.

Ej: Ejecutar sqldeveloper.exe / .sh / .app

Instalación en diferentes sistemas operativos

NOTA SOBRE JAVA (JDK):

Para Windows, puedes descargar la versión "With JDK Included" para mayor facilidad. Para macOS y Linux, asegúrate de tener una versión compatible de Java (JDK) instalada previamente en tu sistema operativo.
