Ventajas Técnicas de VS Code

Entorno Ligero

Consume mucho menos recursos que los IDEs tradicionales orientados a bases de datos.

Alta Extensibilidad

Soporta múltiples lenguajes de programación mediante un vasto ecosistema de extensiones.

Multiplataforma

Brinda una experiencia de desarrollo idéntica y unificada en Windows, macOS y Linux.

ADVERTENCIA PEDAGÓGICA

En Fundamentos de Ingeniería de Software NO se recomienda el uso de VS Code porque incluye potentes herramientas de autocompletado avanzado y asistencia de IA que interfieren negativamente con el aprendizaje fundamental de SQL y la comprensión real de la sintaxis básica.

GitHub Copilot

Genera código SQL completo automáticamente, evitando que el estudiante construya la lógica por sí mismo.

IntelliSense Avanzado

Sugiere la estructura exacta y palabras clave antes de que el estudiante sepa el comando que realmente necesita.

SQL Snippets

Provee plantillas predefinidas de consultas complejas, lo que impide desarrollar la memoria técnica requerida.

Uso Responsable (Objetivo del Curso)

"El objetivo primordial de este curso es aprender SQL desde cero y comprender la teoría relacional, no depender de asistentes de IA. Escribir las sentencias manualmente es la única forma de consolidar tu conocimiento técnico."

Proyección Profesional

Para cursos avanzados y vida laboral, VS Code es una excelente opción profesional.
