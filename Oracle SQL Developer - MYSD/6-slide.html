Instalación Oracle XE

PASOS PRINCIPALES

01

Descargar XE

Descarga Oracle Database Express Edition (18c o 21c). Es la versión ligera y gratuita ideal para estudiantes.

oracle.com/xe

02

Contraseña

Durante la instalación, se te pedirá una contraseña para los usuarios administradores. ¡No la olvides!

Define una simple (ej: oracle)

03

Usuarios

Por defecto se habilitan SYS y SYSTEM. El usuario HR (ejemplos) viene bloqueado.

ALTER USER HR ACCOUNT UNLOCK;
