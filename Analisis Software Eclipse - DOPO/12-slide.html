S4 PRÁCTICA Cierre del Proyecto: Importar de BlueJ a Eclipse

Importación del Proyecto

1

File → Import...

Selecciona la opción General → Existing Projects into Workspace.

2

Seleccionar directorio

Haz clic en Browse, busca la ruta raíz del proyecto original de BlueJ y selecciona Finish.

3

Verificar pestaña Problems

Comprueba que no se reporten errores rojos en la barra inferior de compilación de Eclipse.

4

Ajustar JDK de ser necesario

Si hay errores: Preferences → Java → Installed JREs y añade tu JDK 11/17.

Verificación del Entorno

Proyecto importado correctamente sin fallos de dependencias.
Clase principal ejecutable (Run As → Java Application).
Pruebas JUnit funcionando y pasando en verde (JUnit Test).
Javadoc estructurado y generado (Project → Generate Javadoc...).

    contest.simular();
}
