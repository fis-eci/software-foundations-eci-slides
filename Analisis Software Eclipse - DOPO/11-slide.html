Mutante Superviviente Detectado

PIT cambió el operador de comparación monto > 0 a monto >= 0 en consignar, ¡y ningún assert falló! 
@Test
public void shouldNotConsignarWhenMontoCero() {
    CuentaBancaria cuenta = new CuentaBancaria();
    cuenta.consignar(0.0);
    assertEquals(0.0, cuenta.getSaldo(), 0.01);
}
✔ Solución: El test anterior asegura que consignar 0.0 no altera el saldo de la cuenta, matando de inmediato este mutante alternativo.
