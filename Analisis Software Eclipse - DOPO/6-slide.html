Ejemplo de Método Main indispensable

Recuerda estructurar el punto de entrada para ejecutar las simulaciones requeridas en el taller: 

public static void main(String[] args) {
    Contest contest = new Contest();
    contest.simular();
}
💡 El compilador de Eclipse creará los ejecutables binarios de forma inmediata al vuelo tras cada guardado exitoso (Ctrl + S).
