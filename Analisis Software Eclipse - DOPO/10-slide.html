ANÁLISIS 3 🟢 EclEmma (JaCoCo) — Pruebas de Cobertura

Ejecución de Cobertura

1. Clic derecho sobre la clase Test o paquete → Coverage As → JUnit Test.
2. Al finalizar, Eclipse coloreará de forma automatizada el editor de código.

Leyenda de Colores

Verde Instrucción totalmente ejecutada por las pruebas unitarias.
Rojo Línea NO ejecutada. Cobertura lógica pendiente por verificar.
Amarillo Rama de decisión parcialmente cubierta (ej. 'if' evaluado en true pero no en false).

Simulador Visual de Código Coloreado

public void consignar(double monto) {
    if (monto > 0) {
        saldo = saldo + monto;
    }
}
public void retirar(double monto) {
    if (monto > 0 && monto <= saldo) { // ← Rama parcial
        saldo = saldo - monto;
    } // ← Línea nunca ejecutada por completo
