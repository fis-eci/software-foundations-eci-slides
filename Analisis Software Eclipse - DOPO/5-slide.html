PLUGINS Paso 2: Instalar Plugins via Eclipse Marketplace
Ruta de instalación: Help → Eclipse Marketplace... → Busca cada plugin → Install → Restart Now (uno a uno)
SonarLint
ESTÁTICO
Término de búsqueda: SonarLint
Análisis de diseño en tiempo real

Subraya problemas en el editor mientras escribes, sin ejecutar comandos adicionales. Detecta bugs, vulnerabilidades y code smells de diseño OOP al instante.

PMD Eclipse Plugin
ESTÁTICO
Término de búsqueda: PMD Eclipse Plugin
Análisis de calidad y malas prácticas

Analiza el árbol de sintaxis abstracta para identificar código muerto, bloques catch vacíos, parámetros no utilizados, variables sobreescritas y violación de estándares.

EclEmma / JaCoCo
DINÁMICO
Término de búsqueda: EclEmma
Pruebas de cobertura integradas

Colorea tu base de código en tiempo de ejecución (verde, rojo, amarillo) para mapear qué instrucciones y ramificaciones lógicas están siendo realmente ejercitadas por tus pruebas JUnit.

Pitclipse / PIT
DINÁMICO
Término de búsqueda: Pitclipse
Pruebas de mutación y robustez

Introduce sutiles fallos artificiales de forma automatizada (mutantes) en tu código para verificar que tus pruebas unitarias son capaces de capturar y matar dichos errores de forma oportuna.

⚠️ Recuerda: Es fundamental reiniciar Eclipse después de la instalación de cada plugin para evitar conflictos de runtime.
