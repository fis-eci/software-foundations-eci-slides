ANÁLISIS 1 🔵 SonarLint — Análisis de Diseño en Tiempo Real

Activar y utilizar la vista

1. Ve al menú superior: Window → Show View → Other...
2. Selecciona: SonarLint → On-The-Fly Issues.
3. La pestaña se abrirá mostrando los incidentes ordenados por severidad y tipo.

Flujo de Resolución

1. Identifica el incidente con mayor severidad de la lista.
2. Haz doble clic: Eclipse te llevará exactamente a la línea afectada.
3. Abre la regla de SonarLint para entender por qué es una mala práctica.
4. Aplica la corrección oportuna o documenta tu justificación técnica.
5. Guarda la clase (Ctrl+S) y comprueba que el aviso desaparezca.

Caso de Estudio: CuentaBancaria

REGLA VIOLADA
public void retirar(double monto) {
    if (monto > 0 & monto <= saldo) {  // ← Alerta
        saldo = saldo - monto;
    }
}
Use "&&" instead of "&" (Logical vs Bitwise AND)
CORRECCIÓN DE DISEÑO 
public void retirar(double monto) {
    if (monto > 0 && monto <= saldo) { // ✓ AND lógico
        saldo = saldo - monto;
    }
}
