INSTALACIÓN Paso 1: Instalar Eclipse IDE for Java Developers
Paso 1: Descargar

Obtener el instalador

Entra a eclipse.org/downloads y descarga la versión: Eclipse IDE for Java Developers.

Paso 2: Instalar

Ejecutar el Setup

Abre el archivo descargado. Acepta la ruta de instalación recomendada por defecto, haz clic en Install y luego en Launch.

Paso 3: Workspace

Definir la ruta

Selecciona o crea la carpeta de trabajo principal (Workspace) donde vivirán todos tus proyectos y análisis del curso DOPO.

Paso 4: Verificar

Acceso Exitoso

Asegúrate de que Eclipse abra de forma correcta, sin advertencias de compatibilidad, y muestre la interfaz de bienvenida inicial.

Requisito indispensable: Debes tener instalado previamente el kit de desarrollo Java (JDK 11 o JDK 17) en tu sistema operativo antes de arrancar Eclipse IDE. Compruébalo ingresando la instrucción java -version desde tu terminal de comandos.
