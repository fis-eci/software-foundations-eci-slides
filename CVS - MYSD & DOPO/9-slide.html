¿Qué es Git?

CVS

El Concepto (Teoría)

Es la idea abstracta de registrar cambios, crear versiones y gestionar ramas. Es el "qué" queremos lograr (organización, historial).

Filosofía Metodología Estrategia
VS

Git

La Herramienta (Práctica)

Es el software concreto que instalas en tu computador para implementar CVS. Es el "cómo" lo logramos mediante comandos en terminal.

Software Comandos Terminal

¿Cómo funciona? Git trabaja localmente en tu máquina (sin internet) para gestionar tus versiones privadas. Para colaborar, se conecta a plataformas remotas como GitHub, que alojan tu código en la nube.
