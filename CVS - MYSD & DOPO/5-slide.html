S1: TEORÍA

Componentes de un CVS

Repositorio Local

Tu Computador

Espacio Privado: Solo tú puedes ver y modificar los archivos aquí.

Experimentación: Puedes probar cambios sin miedo a romper el proyecto de los demás.

Historial Completo: Tienes una copia de todas las versiones en tu máquina.

Velocidad: Todo ocurre instantáneamente sin necesitar internet.

Sincronización

Repositorio Remoto

La Nube (GitHub)

Espacio Compartido: Punto de encuentro para todo el equipo.

Verdad Oficial: Contiene la versión "estable" y consolidada del proyecto.

Respaldo: Protege el código si tu computador se daña.

Colaboración: Permite revisar (Pull Request) y fusionar el trabajo de otros.

Ruta de Aprendizaje: Primero domina el Repositorio Local antes de sincronizar con el Repositorio Remoto.
