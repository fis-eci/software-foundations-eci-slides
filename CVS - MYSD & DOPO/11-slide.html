Configuración e Inicio

1. Configurar Identidad Obligatorio

Antes de empezar, debes decirle a Git quién eres. Esto es vital para que tus cambios en GitHub aparezcan con tu nombre y foto correctos. Se hace una sola vez por computador.

# Configura tu nombre completo real $ git config --global user.name "Andersson David Sánchez Méndez" # Configura tu correo institucional $ git config --global user.email "andersson.sanchez-m@mail.escuelaing.edu.co"
--global Aplica para todos tus proyectos en este PC
2. Iniciar Repositorio (Local) Opción A

Úsalo cuando empiezas un proyecto desde cero en tu computador y quieres activar el control de versiones en esa carpeta. Crea la carpeta oculta .git.

# Transforma la carpeta actual en un repositorio Git $ git init
Nota: Ejecutar dentro de la carpeta del proyecto
3. Clonar Repositorio (Remoto) Opción B

Úsalo para descargar un proyecto que ya existe en GitHub. Copia todo el historial y configura automáticamente la conexión remota (origin).

# Descarga una copia completa desde GitHub $ git clone https://github.com/usuario/proyecto.git
<url> Dirección HTTPS del repositorio en GitHub
