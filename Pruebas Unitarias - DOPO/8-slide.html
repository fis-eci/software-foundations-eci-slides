}

2
Casos Nominales
HAPPY PATH

El escenario ideal: consignar un monto válido aumenta el saldo correctamente.

public void consignar(double m) {

  this.saldo += m; // Implementación inicial

}

3
Caso de Borde
EDGE CASE

¿Qué pasa si consigno 0? El saldo no debería cambiar. shouldNotAlterar...

@Test void shouldNotAlterarSaldoWhenConsignarCero() {

  cuenta.consignar(0.0); assert(50.0, cuenta.getSaldo());

}

4
Bug Fixing
BUG FOUND

Bug: Consignar negativo resta saldo. Solución: if (m > 0)

@Test void shouldNotPermitirMontoNegativo() {

  cuenta.consignar(-20.0); assert(100.0, cuenta.getSaldo());

}
